Why Javascript

Javascript has become one of the most utilized programing languages out there, it is no longer just a programming language that exist just on web pages to give it a little boost in appearance and performance.

Javascript has grown into a major player in regular applications and back end applications thanks to advancement in Javascript rendering, in large part thanks to Google's development of the Javascript V8 engine, that has led to the development of Node js that brings that engine into desktop computing and backend web development/
